# Insurance-Analytics-and-Risk-Assessment
Comprehensive insurance analytics project focused on customer churn, claims behavior, payment trends, feature engineering, and risk assessment using Python and data visualization.

Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook


Project Workflow
1. Data Sampling & Validation
2. Exploratory Data Analysis
3. Univariate & Bivariate Analysis
4. Data Cleaning
5. Statistical Analysis
6. Feature Engineering
7. Trend Analysis
8. Business Insight Generation


Dataset Information
The project uses multiple insurance-related datasets including:
- Customer Policies
- Claims Data
- Payments Data
- Medical Checkups Data

The datasets contain information related to premiums, claims, churn behavior, payment activity, medical risk indicators, and customer policy details.
